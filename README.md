# OpenClaw AI Steward - Stitch Design

Google Stitch 导出的 OpenClaw AI 管家 UI 设计稿，基于 Material Design 3 色彩体系。

## 页面一览

| 页面 | 路径 | 说明 |
|------|------|------|
| 商城 | [pages/shop/index.html](pages/shop/index.html) | OpenClaw Shop |
| 管家 | [pages/butler/index.html](pages/butler/index.html) | AI 管家主页 |
| 工具 | [pages/tools/index.html](pages/tools/index.html) | 工具箱 |
| 我的 | [pages/me/index.html](pages/me/index.html) | 个人中心 |

每个页面文件夹包含：
- `index.html` — 完整的独立 HTML 页面（内联 Tailwind CDN）
- `screenshot.png` — 设计截图

## 设计规范

详见 [design/DESIGN.md](design/DESIGN.md)，包含 Material Design 3 色彩 Token 和排版规范。

## 技术栈

- Tailwind CSS (CDN)
- Material Symbols Outlined 图标
- Hanken Grotesk + Geist 字体
- 自定义 Material Design 3 色彩配置

## 预览

直接在浏览器中打开任意 `index.html` 即可查看设计效果。
